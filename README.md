
# Sistema de Fila by Sling

## Sobre o Projeto

Sistema de gerenciamento de filas para controle de atendimento desenvolvido pela Sling Soluções de Mercado.

## Tecnologias utilizadas

Este projeto é construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
